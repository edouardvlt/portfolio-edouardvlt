Observabilité Totale

  • Grafana & Prometheus Métriques long terme, surveillance CPU/RAM/Disque optimisée pour 4 Go de RAM.
  • Logs en Temps Réel (Dozzle) Visualisation instantanée des logs conteneurs via interface Web sécurisée.
  • Alerting Proactif Notifications SMTP via iCloud en cas d'indisponibilité d'un service.

Stratégie de Défense

L'accès aux interfaces critiques (Portainer, Dozzle) est strictement verrouillé.

DNS Challenge

Certificat SSL Wildcard (*.edouard-vallet.fr) généré sans ouvrir le port 80, via l'API Cloudflare.

Access Lists

Authentification Nginx supplémentaire devant les services sensibles.

Isolation Réseau

Flux Portainer forcé en HTTP interne (port 9000) derrière le SSL du Proxy pour éviter les conflits.

Full Strict

Chiffrement de bout en bout forcé entre le client, Cloudflare et le serveur Acer.
